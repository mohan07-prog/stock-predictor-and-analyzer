# Stock Analysis & Prediction Tools

Two small tools for stock market analysis:

1. `stock_predictor.py`  
   → Machine Learning stock price prediction using MLPRegressor (Artificial Neural Network)  
   → Downloads 7 years of HDFC Life data from Yahoo Finance

2. `mian.py` (probably typo → should be `main.py`?)  
   → Drag & drop CSV stock data analyzer with SMA + trend line visualization  
   → Built with tkinter + tkinterdnd2

## Requirements

pip install yfinance pandas numpy scikit-learn matplotlib tkinterdnd2
